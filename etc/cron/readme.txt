To use crontabs, please use the cron manager to setup your crontabs. Alternatively, manually create a file called 'root' and place your required items in there.

# │ │ │ │ │

# │ │ │ └────────── month (1 - 12)
# │ │ └─────────────── day of month (1 - 31)
# │ └──────────────────── hour (0 - 23)
# └───────────────────────── min (0 - 59)
